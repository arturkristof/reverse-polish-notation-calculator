# reverse-polish-notation calculator
Project for university from January 2016

This program calculates formulas using [Reverse Polish Notation](https://en.wikipedia.org/wiki/Reverse_Polish_notation "Wikipedia").  

Example input file:
>2+2  
>(2+3)*5  
>((2+7)/3+(14-3)*4)/2  
>2 2 +  
>2 3 + 5 *  
>2 7 + 3 / 14 3 - 4 * + 2 /
